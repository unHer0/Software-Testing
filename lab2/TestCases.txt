Тест-кейс № 1. Ввод одинакового пункта отправления и прибытия.
Шаги:
1. Зайти на главную страницу
2. В поле "Откуда" ввести город вылета (Минск)
3. В поле "Куда" ввести тот же город, что и ранее (Минск)
4. Выбрать дату полета 
5. Нажать кнопку "Поиск рейса"
Ожидаемый результат: появление модального окна с сообщением "Ошибка. Пункт отправления не может совпадать с пунктом назначения. Измените один из вариантов".


Тест-кейс № 2. Поиск билета на год вперёд.
Шаги:
1. Зайти на главную страницу

Билет не может быть заказан за год и более до даты вылета.

Тест-кейс № 5. Поиск одного билета для младенца (меньше 2 лет).
Шаги:
1. Зайти на главную страницу
2. Нажать кнопку "Кабина"
3. Установить количество мест  для младенцев - 1, при помощи "+"
4. Установить количество взрослых пассажиров - 0, при помощи "-"
5. Нажать кнопку "Поиск рейса"
Ожидаемый результат: появление предупреждения в меню выбора кол-ва пассажиров "Выберите минимум 1 пассажира, не являющегося младенцем.".
При нажатии кнопки "Поиск рейса" - появление модального окна с сообщением "Ошибка. Выберите пассажира".
